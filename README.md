## How to start

download VScode
download Python
download git

download homebrew
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew --version
```

download uv
```bash
brew install uv
uv --version
```

# download 
```bash
uv pip install pytesseract pillow
```
